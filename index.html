CSIS1430 Reflection

The course was a great introduction to HTML, CSS, and JS. As an introductory course, it did a great job in teaching the basics and building a solid foundation for advanced topics.

I found the flipped classroom model to be quite valuable. The pre lectures allowed me to learn the basics at my own pace and solidify them in class. I enjoyed the class lectures as they were shaped by the students' questions and interests. This learning model focuses on exploration and asking questions which is an important skill for web developers.

I'm glad I took this course. I've come far since the start of the course, and this portfolio is proof of that. I hope to continue learning how to build web things, regardless as a hobby or a career path.
